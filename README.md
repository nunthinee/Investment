# คู่มือนำ "สมุดปันผล" ขึ้น Vercel ฟรี ทีละขั้นตอน

ผลลัพธ์สุดท้าย: คุณจะได้เว็บของตัวเองที่ URL เช่น `https://dividend-app-xxx.vercel.app` เปิดได้จากมือถือ/คอมทุกที่ ราคาหุ้นอัปเดตอัตโนมัติทุก 60 วินาที ฟรี 100% ไม่ต้องสมัคร API key ใด ๆ (ระบบดึงราคาจาก Yahoo Finance ผ่าน serverless function ที่แถมมาในโปรเจกต์นี้แล้ว)

ใช้เวลารวมประมาณ 15 นาที ทำจากมือถือได้ แต่แนะนำทำบนคอมพิวเตอร์จะง่ายกว่า

---

## ขั้นที่ 1 — สมัคร GitHub (ที่เก็บโค้ด)

1. เข้า https://github.com กด **Sign up**
2. กรอกอีเมล ตั้งรหัสผ่าน ยืนยันอีเมล — เสร็จ

## ขั้นที่ 2 — สร้าง Repository และอัปโหลดไฟล์ (ไม่ต้องใช้คำสั่งอะไรเลย)

1. ล็อกอิน GitHub แล้วกดปุ่ม **+** มุมขวาบน → **New repository**
2. ตั้งชื่อ `dividend-app` เลือก **Public** → กด **Create repository**
3. ในหน้าที่เปิดขึ้นมา กดลิงก์ **uploading an existing file**
4. แตกไฟล์ zip ที่ได้จากแชทนี้ก่อน แล้ว**ลากไฟล์และโฟลเดอร์ทั้งหมดข้างใน** (package.json, vite.config.js, index.html, โฟลเดอร์ api, โฟลเดอร์ src, README.md) ไปวางในหน้าอัปโหลด
   - สำคัญ: ลาก "ของข้างในโฟลเดอร์ dividend-app" ไม่ใช่ลากตัวโฟลเดอร์ dividend-app ทั้งก้อน เพื่อให้ package.json อยู่ชั้นบนสุดของ repo
5. เลื่อนลงล่าง กด **Commit changes**
6. ตรวจว่าหน้า repo มีไฟล์ครบ: `package.json`, `vite.config.js`, `index.html`, โฟลเดอร์ `api/`, โฟลเดอร์ `src/`

## ขั้นที่ 3 — สมัคร Vercel และ Deploy

1. เข้า https://vercel.com กด **Sign Up** → เลือก **Continue with GitHub** (ใช้บัญชีจากขั้นที่ 1)
2. อนุญาตให้ Vercel เข้าถึง GitHub เมื่อระบบถาม
3. ที่หน้า Dashboard กด **Add New…** → **Project**
4. จะเห็นรายชื่อ repo ของคุณ — กด **Import** ที่ `dividend-app`
5. หน้า Configure Project: Vercel จะตรวจพบเป็น **Vite** อัตโนมัติ ไม่ต้องแก้อะไร → กด **Deploy**
6. รอ 1–2 นาที จนขึ้นพลุฉลอง 🎉 → กด **Visit** หรือคัดลอก URL ที่ได้

## ขั้นที่ 4 — ใช้งานบนมือถือเหมือนแอปจริง

1. เปิด URL ที่ได้ในเบราว์เซอร์มือถือ
2. iPhone: กดปุ่ม Share → **Add to Home Screen** / Android: เมนู ⋮ → **เพิ่มลงในหน้าจอหลัก**
3. จะได้ไอคอนแอปบนหน้าจอ เปิดใช้ได้ทันที ราคาอัปเดตเองทุก 60 วินาที

## การแก้ไขข้อมูลในอนาคต (เช่น เพิ่มหุ้น/อัปเดตปันผลปีใหม่)

1. เข้า repo บน GitHub → เปิดไฟล์ `src/App.jsx` → กดไอคอนดินสอ (Edit)
2. แก้ตัวเลขในตัวแปร `TH_STOCKS` / `US_STOCKS` ตามรูปแบบเดิม
3. กด **Commit changes** — Vercel จะ build และอัปเดตเว็บให้อัตโนมัติภายใน ~1 นาที

## ปัญหาที่พบบ่อย

- **Deploy แล้ว error "No package.json"** → ไฟล์อยู่ลึกไป 1 ชั้น (ลากทั้งโฟลเดอร์เข้าไป) ให้ลบ repo แล้วอัปโหลดใหม่ตามข้อ 2.4
- **ราคาไม่ขึ้น LIVE** → รอ 1 นาทีให้รอบอัปเดตแรกทำงาน หรือกด "รีเฟรชตอนนี้" ที่หัวเว็บ ถ้ายังไม่มา เปิด `https://โดเมนของคุณ/api/quotes?symbols=PTT.BK` ดูว่าได้ตัวเลขไหม
- **อยากใช้ Netlify แทน** → ได้เช่นกัน แต่ต้องย้ายไฟล์ `api/quotes.js` ไปเป็น Netlify Function (โครงสร้างต่างกันเล็กน้อย) — แนะนำ Vercel เพราะใช้ไฟล์ชุดนี้ได้ทันที

## ข้อจำกัดที่ควรรู้

- ราคาหุ้นไทยจาก Yahoo Finance ดีเลย์ ~15 นาที (หุ้นสหรัฐฯ ใกล้ realtime) — เพียงพอสำหรับติดตามพอร์ต แต่ห้ามใช้ตัดสินใจส่งคำสั่งซื้อขายรายวินาที
- ต้องการข้อมูลไทยแบบทางการ/realtime แท้: สมัคร Settrade Open API (ผ่านโบรกเกอร์ที่มีบัญชี) หรือ SET SMART Marketplace แล้วแก้ `api/quotes.js` ให้ชี้ไปที่ API นั้นแทน
- ข้อมูลปันผลย้อนหลังในแอปเป็นค่าโดยประมาณเพื่อการศึกษา ตรวจสอบจริงที่ set.or.th ก่อนลงทุนเสมอ

## หน้า Forex & ทองคำ (เพิ่มใหม่)

- **ราคาสด + แนวโน้ม**: USD/THB, EUR/USD, GBP/USD, USD/JPY และทองคำ (Gold Futures) อัปเดตทุก 60 วินาที พร้อมป้ายแนวโน้ม ขาขึ้น/ขาลง/Sideways (คำนวณจาก SMA20 + โมเมนตัม 1 เดือน)
- **เวลาตลาด**: แสดงเซสชันซิดนีย์/โตเกียว/ลอนดอน/นิวยอร์กเป็นเวลาไทย พร้อมสถานะเปิด-ปิดแบบเรียลไทม์ (Forex เปิด จ. 04:00 – ส. 04:00 เวลาไทย)
- **ตั้งเตือนราคา**: เลือกคู่เงิน/ทอง + ขึ้นถึง/ลงถึง + ราคาเป้าหมาย → เมื่อถึงเป้าระบบเด้ง Browser Notification พร้อมเสียง ค่าเตือนบันทึกถาวรในเครื่อง (localStorage)
  - ข้อจำกัด: การเตือนทำงานขณะเปิดหน้าเว็บค้างไว้ (สลับแท็บ/พับจอได้ แต่ห้ามปิดแท็บ) — ถ้าต้องการเตือนแม้ปิดแอป ต้องต่อยอดเป็น Push Notification ผ่าน service worker หรือส่งเข้า Telegram Bot
- **ข่าว ForexFactory**: ปฏิทินข่าวเศรษฐกิจสัปดาห์นี้จากฟีดทางการของ ForexFactory กรองตามระดับผลกระทบ แสดงเวลาไทย คาดการณ์ และค่าครั้งก่อน

ไฟล์ที่เกี่ยวข้อง: `src/ForexTab.jsx`, `api/fx.js`, `api/ffnews.js` — ถ้าอัปโหลด repo ไปแล้ว ให้เพิ่ม 3 ไฟล์นี้ + แทนที่ `src/App.jsx` ด้วยเวอร์ชันใหม่ แล้ว Commit, Vercel จะอัปเดตให้เอง
